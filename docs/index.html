Para leigos

O que é o score?

O score é uma nota de triagem quantitativa. Ele compara cada ativo com os demais ativos do mesmo universo no dia da coleta. Quanto maior o score, melhor o ativo ficou posicionado dentro dos critérios usados pelo robô.

Ele não diz que o ativo é “barato para sempre” nem garante retorno. Ele ajuda a separar rapidamente quais ativos merecem uma análise mais profunda.

Importante

O que o score não mede

O modelo não substitui análise qualitativa. Ele não avalia governança em profundidade, risco jurídico, mudança estrutural de negócio, qualidade da gestão, eventos extraordinários ou notícias recentes.

Use o score como filtro inicial, não como recomendação final.

Como o ranking é calculado

O robô coleta os dados, aplica filtros mínimos, normaliza os indicadores por comparação relativa e calcula uma pontuação ponderada. O Top 20 mostra os ativos mais bem colocados dentro desse recorte.
