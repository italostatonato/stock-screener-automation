Como o score funciona

Leitura rápida

Um score 82 significa que o ativo está melhor posicionado que boa parte do universo analisado nos indicadores ponderados. A metodologia usa percentis: maior é melhor para DY, ROA, liquidez e patrimônio; menor é melhor para P/VP, P/L, EV/EBITDA e volatilidade.
